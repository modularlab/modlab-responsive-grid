MODLAB Responsive Grid

This is a demo of a 3 breakpoint (Desktop/Tablet/Mobile), percent-based responsive grid, which relies solely on CSS and media queries. The only javascript involved with this grid is to force Internet Explorer 8 to recognize media queries. By utilizing a variety of classes, unique to each breakpoint, this grid allows for flexibility and difference between Desktop, Tablet, and Mobile layouts.
